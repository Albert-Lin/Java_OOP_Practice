# Java 物件導向程式設計練習
"物件導向"是一種程式撰寫跟運作的模式跟原理，
目前多數的程式語言都是以此設計模式在進行(當然有些例外)。

本專案主要目標在於讓讀者可以藉由各階段的範例跟說明，
了解並可以讀懂以物件導向為基礎的程式碼，其主題與件數如下表:


主題         | 簡述
------------|-------
Env         | 環境設定
Hello World | 名詞定義與概念說明
Type        | 介紹基礎型別
Loop        | 介紹常用迴圈
Array       | 介紹陣列
Class       | 介紹類別
Extends     | 介紹類別的繼承 & 多型(介面)
Generics    | 介紹泛型類別
ArrayList   | 介紹鏈結串列
